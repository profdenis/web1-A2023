Alertes avec Boostrap

Une alerte est parfaite pour avertir un utilisateur d'une mise à jour, d'une erreur ou d'un avertissement. Généralement, c'est un traitement en cours ou une action qui lance l'alerte. Comme nous sommes limités dans l'utilisation de javascript, l'alerte doit être présente par défaut. De plus, les alertes sont parfaites avec les icônes.
