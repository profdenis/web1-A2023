Tooltips avec Boostrap

Un tooltip nous permet de faire apparaitre une description quand la souris est par dessus un mot.

Ça peut aussi être sur un bouton, un span, un hyperlien, ... L'important est d'initialisé le tout avec deux lignes de javascript et après on peut mettre le texte qu'on veut dans l'attribut data-bs-title

Popover avec Boostrap

Si vous voulez afficher quelques choses de plus complexe comme un titre avec du texte. Il vous faut un popover! Par contre, nous sommes limités par le javascript, alors on ne peut pas faire quelque chose de plus complexe.
